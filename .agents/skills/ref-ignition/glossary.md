# Ignition SCADA Glossary

| Term (EN) | Term (CN) | Definition |
|-----------|-----------|------------|
| **Gateway** | 网关 | Ignition's core Java server process — heart of the platform |
| **Tag** | 标签 | Data point (analog, digital, string) managed by the Tag Engine |
| **Tag Provider** | 标签提供者 | Data source abstraction layer (OPC UA, Modbus, MQTT, Memory, etc.) |
| **Perspective** | 透视 | Web/mobile HMI module (HTML5/React, browser-based, responsive) |
| **Vision** | 视图 | Legacy desktop HMI module (Java Swing) |
| **Designer** | 设计器 | Engineering IDE for project development |
| **Solution Suite** | 解决方案套件 | Module bundling/packaging (5 suites in 8.3) |
| **Upgrade Protection** | 升级保护 | Continuous update support plan |
| **Inductive University** | 感应大学 | Free online training platform (200+ courses) |
| **Integrator Program** | 集成商计划 | 3-tier certified integrator system |
| **Alarm Pipeline** | 报警管道 | Configurable alarm processing flow |
| **Sparkplug** | Sparkplug | MQTT industrial payload specification |
| **OPC UA** | OPC UA | Open Platform Communications Unified Architecture |
| **MQTT** | MQTT | Message Queuing Telemetry Transport protocol |
| **Web-Launch** | 网页启动 | Clients launch from Gateway via browser (no install) |
| **Hub and Spoke** | 中心辐射架构 | Central Gateway + remote edge Gateways for multi-site |
| **Scale-Out** | 水平扩展 | Multi-Gateway cluster for high concurrency |
| **Jython** | Jython | Python 2.7 implementation on Java (Ignition's scripting engine) |
| **Transaction Group** | 事务组 | Configurable data acquisition/forwarding from tags to SQL |
| **Named Query** | 命名查询 | Predefined parameterized SQL query callable from any module |
| **UDT** | 用户定义类型 | User-Defined Type — reusable tag structure template |
| **Power Chart** | 强力图表 | High-performance time-series chart (million-point render) |
| **Easy Chart** | 简易图表 | Basic trend chart (Vision only) |
| **Deadband** | 死区 | Data compression threshold (value must change by X% to record) |
| **Swinging Door** | 旋转门 | Advanced data compression algorithm for trending |
| **Security Level** | 安全级别 | Hierarchical permission model replacing traditional roles |
| **Shelving** | 搁置 | Temporary alarm suppression (time-based or permanent) |
| **Edge** | 边缘版 | Lightweight Ignition for edge devices (Raspberry Pi, industrial gateways) |
| **Maker Edition** | 创客版 | Free Ignition for non-commercial/education use |
| **EAM** | 企业管理 | Enterprise Administration Module — centralized multi-Gateway management |
