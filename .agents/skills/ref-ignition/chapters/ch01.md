## 1.2 Core Value Proposition

Ignition's positioning as a **next-generation Industrial Application Platform** with four differentiators:

| Differentiator | Description |
|---------------|-------------|
| **Unlimited Licensing** | Per-server license, unlimited clients/tags/projects |
| **Web-Native** | All engineering and runtime via browser — zero client install |
| **Cross-Platform** | Gateway on Linux/Windows/macOS |
| **Modular** | Free core, buy modules as needed |
| **Open Standards** | OPC UA, MQTT Sparkplug, SQL, REST API first-class |

Slogan: "The New SCADA" — replacing 1990s-era products (Wonderware, iFIX, WinCC).

## 1.4 Licensing Model

### License Tiers

| Tier | Model | Use Case |
|------|-------|----------|
| **Ignition Platform** | Perpetual (one-time) | Core platform, foundation for all modules |
| **Ignition Edge** | Perpetual | Edge devices (limited: 1 project, 1 Tag Provider, no direct DB) |
| **Cloud Edition** | Usage-based | Cloud deployment |
| **Maker Edition** | Free | Personal, education, non-commercial |

### Solution Suites (Ignition 8.3)

| Suite | Modules | Function |
|-------|---------|----------|
| Application Building | Perspective + Reporting | Mobile responsive HMI + reports |
| Industrial Historian | Historian + SQL Historian | High-performance time-series storage |
| Enterprise Integration | MongoDB, Kafka, MQTT Transmission, MQTT Engine | Enterprise data integration + IIoT |
| Alarm Management | Alarm Notification + Voice + SMS + Twilio | Advanced alarm notification |
| DataOps | SQL Bridge + Web Dev + Event Streams | Data orchestration + custom web |

### Price Reference

- **Platform Base**: ~$10,000-$20,000 (one-time)
- **Solution Suites**: ~$5,000-$15,000 each (unconfirmed)
- **Individual Modules**: ~$1,000-$5,000 each (unconfirmed)
- **Ignition Edge**: ~$1,500-$3,000 (unconfirmed)
- **Support + Upgrade Protection**: 15%-20%/year of license cost

**Key**: Large projects save 50%-80% vs traditional SCADA. 5000-tag + 10-client traditional = $50k-$100k; Ignition = $10k-$30k.

### License Management

- 6-digit key, activate via Gateway web UI
- Deactivate and transfer freely
- 30-day full trial (resettable twice)
- Edge: 35-day local data cache built-in
