## 3.2 Alarm Management (ISA 18.2)

- **Definition**: Tag threshold alarms (high/low/high-high/low-low/deviation/rate-of-change)
- **Alarm Pipeline**: Configurable flow (Filter → Split → Notify → Store)
- **Notification**: Email, SMS, voice (Alarm Notification Suite), Teams/Slack (Webhook)
- **Acknowledgment**: With comment, electronic signature
- **Archiving**: Auto-store to SQL database
- **Shelving**: Temporary or permanent alarm suppression
- **Grouping**: By area, equipment, priority

## 3.3 Trending & Historian

| Component | Description |
|-----------|-------------|
| **SQL Historian** | Tag data to standard SQL (MySQL/PostgreSQL/SQL Server) |
| **Ignition Historian** | Dedicated high-perf time-series engine (10-100x faster than SQL) |
| **Data Compression** | Deadband + Swinging Door compression |
| **Data Backfill** | Cache on disconnect, backfill on reconnect |
| **Power Chart** | Client-side chart: million-point render, zoom, pan |
| **Easy Chart** | Basic trend (Vision only) |

## 3.4 Reporting

- Based on JasperReports (open-source Java reporting)
- Output: PDF, Excel, HTML, CSV
- Scheduled generation + email delivery
- Parameterized reports
- Subreport support

## 3.5 Recipe Management

- Recipe = set of tag values
- Storage: SQL database
- Operations: Download (write to PLC), Upload (read from PLC), Compare
- Electronic signature for security

## 3.6 User & Role Management

- Local users + AD/LDAP + Federated Identity (SAML/OAuth/OpenID)
- Security Level hierarchy (inheritance)
- Multi-identity-provider user sources

## 3.7 Redundancy

- Active-Standby with heartbeat
- Automatic failover
- History data synchronization
- Requires two licenses (primary + backup)

## 3.8 Multi-Language Support

- Vision: Translation Manager (key-value)
- Perspective: Built-in i18n with dynamic language switching
- Translation file: JSON key-value pairs
