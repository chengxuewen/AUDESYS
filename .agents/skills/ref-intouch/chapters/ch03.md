# Ch03: 功能概览 — Features

## 3.1 可视化（Visualization）

| 功能 | 说明 |
|------|------|
| **矢量图形** | 内置图元，支持渐变填充、透明度、旋转、缩放 |
| **位图/图像** | BMP、JPG、PNG、SVG（2023+） |
| **符号库** | 预置 ISA 标准符号（泵、阀、电机、储罐） |
| **工业图形** | ArchestrA Symbol Editor 创建，对象化封装 |
| **Wizard** | 预配置复合对象（报警浏览器、趋势控件） |
| **动画链接** | 值显示、颜色变化、位置移动、尺寸缩放、可见性、闪烁 |
| **Symbol Wizard** | 可配置参数的 Wizard 模板 |

### Industrial Graphics 示例

```
PumpSymbol:
  - Visual Elements:
    - PumpBody (Ellipse, fill color = if Running then Green else Gray)
    - DirectionArrow (visible = Running)
    - SpeedLabel (text = Speed value + " RPM")
  - Custom Properties:
    - Running: Boolean
    - Speed: Real
    - LabelText: String
```

## 3.2 报警管理（Distributed Alarm System）

### 报警架构

分布式报警系统（Provider-Consumer 模型）：

- **Alarm Provider**: InTouch 本地 Tag 报警 / Application Server 对象报警 / 外部 Provider
- **Alarm Manager**: Summary Alarms（当前活动报警）+ Historical Alarms（内存缓冲区）
- **Alarm Consumer**: Alarm DB Logger / Alarm Printer / Alarm Viewer ActiveX

### 报警类型

| 类型 | 说明 |
|------|------|
| **Value Alarm** | 基于 Tag 值的阈值判断（HIHI/HI/LO/LOLO 四级） |
| **Deviation Alarm** | 目标值与实际值的偏差 |
| **Rate-of-Change Alarm** | 变化速率超限 |
| **Event** | Tag 值变化来源记录 |

### 分布式报警核心特性

| 特性 | 说明 |
|------|------|
| **Provider-Consumer 解耦** | 报警产生节点与消费节点独立部署 |
| **分布式报警记录** | 多节点报警构成逻辑统一集合 |
| **毫秒级 UTC 时间戳** | Provider 端生成，跨时区 SOE 重现 |
| **无缝故障转移** | Backup Provider 在主节点断开时自动接管 |
| **过滤与抑制** | 按类/Tag/组过滤，节点级抑制 |
| **远程确认** | 从任何网络节点确认报警 |

## 3.3 历史数据与趋势（Historian & Trending）

| 功能 | 说明 |
|------|------|
| **InTouch 本地历史** | LGH 文件，可配置死区和间隔 |
| **AVEVA Historian** | SQL Server Block 技术，高速存储 |
| **Historian Client** | 交互式趋势图：缩放、平移、游标读数 |
| **History Playback** | 历史回放 — 以录像形式回放历史 HMI 画面 |
| **分布式历史系统** | 从网络中任何节点检索历史数据 |

> **Historian Block 技术**：将连续时间段的 Tag 数据压缩为块，包含起始值、结束值、时间范围和质量信息，大幅减少 I/O 和磁盘占用。

## 3.4 安全与审计

- OS 集成认证（AD/LDAP）
- ArchestrA RBAC（Galaxy 级）
- Tag 级安全（最小访问级别）
- 区域安全（按工厂区域划分权限）
- 电子签名（FDA 21 CFR Part 11 合规）
- TLS 加密（节点到节点通信）
- 集中补丁管理

## 3.5 Recipe Management（配方管理）

| 功能 | 说明 |
|------|------|
| **Recipe Template** | CSV 电子表格定义配方模板 |
| **Unit 映射** | 配方参数映射到生产单元 Tag |
| **Recipe 操作** | Load / Save / Delete |
| **安全控制** | $AccessLevel 控制配方加载权限 |

## 3.6 SPC（统计过程控制）

InTouch SPC Pro 提供：
- 控制图（X-bar、R、S、P、NP、C、U、EWMA）
- 过程能力分析（Cp、Cpk、Pp、Ppk）
- 报警规则（Western Electric 规则、Nelson 规则）

## 3.7 冗余与高可用

| 冗余级别 | 故障转移时间 |
|----------|------------|
| AppEngine Primary/Backup | ~15 秒 |
| Galaxy Repository (SQL Server HA) | 取决 SQL 配置 |
| Historian (Tier-2 汇总 + 本地缓冲) | < 30 秒 |
| I/O Server (多 DA Server) | ~5-10 秒 |
| 网络冗余 (双 NIC) | 实时 |

## 3.8 多语言支持

运行时语言切换：德语、日语、法语、简体中文等，包括静态文本翻译、Tag 报警消息翻译、错误提示翻译。
