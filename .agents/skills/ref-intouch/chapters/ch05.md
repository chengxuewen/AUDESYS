# Ch05: 市场定位 — Market Position

## 5.1 竞争格局总览

| 竞争对手 | 开发商 | 核心优势 |
|----------|--------|---------|
| **Ignition** | Inductive Automation | 无限许可、Web 原生、跨平台 |
| **WinCC** | Siemens | SIMATIC PLC 紧耦合、TIA Portal 集成 |
| **FactoryTalk View** | Rockwell Automation | Allen-Bradley PLC 紧耦合、PlantPAx 体系 |
| **iFIX** | Emerson (GE Digital) | 分布式架构、水/电/油气行业积累 |
| **GENESIS64** | Iconics | 原生 64 位、OData REST API、3D 可视化 |
| **Citect SCADA** | AVEVA（同一家） | 高性能驱动、采矿/金属行业 |

## 5.2 与 Ignition 的直接对比（最重要的竞争关系）

| 维度 | InTouch / System Platform | Ignition |
|------|--------------------------|----------|
| **许可证模型** | 历史：按 Tag 阶梯。2024+：Unlimited | 按服务器许可，不限 Tag |
| **价格参考** | Unlimited Professional ~$20,000 | Platform Base ~$10,000-20,000 |
| **Web 客户端** | InTouch OMI（HTML5）+ RDS | Perspective（原生 HTML5 React） |
| **工程 IDE** | WindowMaker + ArchestrA IDE（桌面） | Ignition Designer（Web 浏览器） |
| **脚本语言** | QuickScript + .NET（C#） | Python (Jython 2.7) |
| **数据模型** | 对象化（ArchestrA）+ 向后兼容 Tag | 统一 Tag Provider 模型 |
| **安装依赖** | Windows Only + SQL Server | Java + 数据库（多平台） |
| **MQTT 支持** | 新增（2023+） | 原生 Sparkplug B |

> 核心问题：Ignition 的无限许可 + Web 原生架构正在从 InTouch 手中夺取新增市场份额。

## 5.3 市场定位分析

| 场景 | 推荐方案 | 原因 |
|------|---------|------|
| 已有 InTouch 历史系统 | **保持 InTouch** | 迁移成本极高 |
| 新建大型企业流程 SCADA | InTouch + System Platform | ArchestrA 对象模型对多站点标准化有效 |
| 新建中小型系统 | **Ignition**（通常更优） | 更低的许可成本、更现代的 Web 架构 |
| 制药/生命科学（21 CFR Part 11） | InTouch + AVEVA Recipe Mgt | 全栈合规 |
| 仅需嵌入式 HMI（OEM） | InTouch Edge HMI | 紧凑安装、低成本 |

## 5.4 定价竞争力分析

**历史痛点 vs 2024 年改革**：

| 历史痛点 | 2024 年改革 |
|----------|-----------|
| 按 Tag 阶梯定价，超限需"丢弃旧许可" | Unlimited 模型：无限 Tag |
| 开发环境需购买额外许可 | **开发工具免费** |
| 许可升级是"可弃成本和替换" | 订阅制可选 |
| 全产品栈许可昂贵 | InTouch Unlimited 已包含 Historian + Reporting |

**TCO 对比（估算）**：

| 场景 | InTouch（传统） | InTouch Unlimited | Ignition |
|------|---------------|-------------------|----------|
| 500 Tag 单机 | ~$5,000-8,000 | ~$12,000（Standard） | ~$10,000 |
| 5000 Tag + 10 客户端 | ~$50,000-100,000 | ~$20,000（Professional） | ~$20,000-30,000 |
| 企业级多站点 | 定制报价 | ~$20,000/2 节点 | ~$50,000+ |
