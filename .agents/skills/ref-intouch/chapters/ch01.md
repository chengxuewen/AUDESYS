## 1.2 历史沿革与关键里程碑

InTouch 的开发商变迁反映了工业软件行业并购整合的典型路径：

| 时间 | 里程碑 |
|------|--------|
| **1987 年** | Wonderware 在加州 Irvine 成立 |
| **1989 年** | InTouch 1.0 发布 — 工业史上第一个基于 Windows 的 HMI/SCADA 软件 |
| **1990 年代** | 发布 FactorySuite 集成套件（InTouch、InControl、IndustrialSQL Server、InTrack、InBatch） |
| **1998 年** | Wonderware 被英国 Siebe plc 收购 |
| **1999 年** | Siebe 与 BTR plc 合并为 Invensys plc |
| **2014 年** | 施耐德电气以 34 亿英镑收购 Invensys |
| **2018 年** | Schneider 工业软件业务与 AVEVA 合并，Wonderware 品牌正式退役 |
| **2021 年** | AVEVA 以 50 亿美元收购 OSIsoft（PI System） |
| **2023 年** | Schneider 完成对 AVEVA 的全资收购，AVEVA 从伦敦交易所退市 |
| **2024 年** | 发布 InTouch Unlimited 新定价策略，取消 Tag 数量限制 |

## 1.3 历史地位与技术影响力

1. **首创 Windows HMI 范式**：定义了"图形化工厂操作界面"的标准。WindowMaker（开发环境）+ WindowViewer（运行环境）的 IDE/Runtime 分离设计，影响了其后几乎所有 HMI 产品。
2. **Tag-based 数据模型的行业标准**：Tagname Dictionary 定义了 HMI 数据管理的基本范式。
3. **ArchestrA 面向对象革命**：2000 年代初推出，将传统平面 Tag 模型升级为面向对象的层次化模型。
4. **FactorySuite 集成套件**：行业最早的"一站式"工业软件套件。
5. **最大的 HMI 安装基础**：在全球 HMI/SCADA 市场占据约 1/3 的安装基础。

## 1.4 产品线全景

| 产品 | 定位 | 核心特性 |
|------|------|----------|
| **InTouch HMI** | 经典 PC 端 HMI | Tag-based 可视化，WindowMaker + WindowViewer，QuickScript |
| **InTouch Unlimited** | 全功能 HMI/SCADA（新定价） | 无限 Tag/客户端，含 Historian + Reporting |
| **InTouch Edge HMI** | 嵌入式/设备级 HMI | Windows Embedded，紧凑型 |
| **System Platform** | 企业级 SCADA 平台 | Galaxy 集中管理，对象模板，分布式部署 |
| **OMI** | Web/响应式可视化 | HTML5，跨设备自适应，情境感知 UI |
| **Historian** | 工业时序数据库 | Block 技术，高速存储 |

## 1.5 InTouch vs System Platform 定位差异

| 维度 | InTouch Classic | System Platform |
|------|----------------|-----------------|
| **数据模型** | 平面 Tag 模型 | 面向对象模型（ArchestrA Object） |
| **配置管理** | 本地文件 | Galaxy 集中式数据库 |
| **多工程师协作** | 有限（NAD 手动同步） | 完全（检出/检入） |
| **学习曲线** | 低（1-2 天入门） | 高（建议正式培训） |
| **适用规模** | 单机到中小型系统 | 多节点企业级 |

> 关键须知：即使仅安装 InTouch Classic，也必须安装完整的 System Platform 安装仓库（约 4.5 GB）。

## 1.6 商业许可模型

| 许可模型 | 详情 |
|----------|------|
| **历史模型**（1989-2024） | 永久许可，按 Tag 阶梯定价：100/256/512/1000/3000/8000/32K/64K |
| **当前模型**（2024+） | InTouch Unlimited：无限 Tag + 无限客户端，永久许可和订阅制可选，开发工具免费 |
| **Premier 订阅** | 在 Professional 功能基础上增加 Unlimited Historian |
| **Operations Control** | 仅订阅制，零 Tag/IO/服务器限制 |
| **生命周期** | 完整支持约 5 年 + 有限支持 2 年 + 技术支撑 |
