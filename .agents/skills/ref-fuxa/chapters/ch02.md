## 2.2 Backend: Node.js + Express

- **Runtime**: Node.js 18 LTS (recommended) / Node.js 20+
- **WebSocket**: Backend pushes real-time data to frontend (tag value changes, alarm events) via socket.io
- **REST API**: Complete RESTful API for external system integration
- **Process modes**: Standard (single process) and Daemon (background daemon for embedded/headless)
- **Storage**: Config = SQLite; History = SQLite or InfluxDB; Project files = filesystem (JSON format)

## 2.3 Frontend: Angular SPA

- **Framework**: Angular (version unspecified, likely 15+)
- **SVG Editor**: Browser-native SVG drag-and-drop editor вҖ” FUXA's most unique component
- **Real-time update**: WebSocket receives tag data changes, millisecond-level screen updates
- **Responsive design**: Mobile browser support (not a dedicated mobile app)

## 2.7 Plugin Architecture for Protocol Adapters

Each adapter implements a unified interface (connect, disconnect, read, write). Adding a new protocol only requires implementing the same interface. The adapters are independent modules in `protocols/`.
