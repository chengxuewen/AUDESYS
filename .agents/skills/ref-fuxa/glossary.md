# Glossary — FUXA Terminology

| Term | English | Description |
|------|---------|-------------|
| Tag | Tag | Data point (corresponds to a variable/register on a device) |
| Device | Device | Abstraction of a physical device (PLC, sensor, etc.) |
| Protocol Adapter | Device Driver / Protocol Adapter | Driver that interfaces with a specific communication protocol |
| SVG Editor | SVG Editor | Drag-and-drop screen designer based on browser-native SVG |
| View | View | One HMI monitoring screen page |
| Data Binding | Data Binding | Mechanism that binds Tag values to SVG component properties |
| Conditional Formatting | Conditional Formatting | Component style that changes automatically based on Tag values |
| Daemon Mode | Daemon Mode | Background daemon process execution mode |
| MIT License | MIT License | One of the most permissive open-source software licenses |
| Tag Engine | Tag Engine | Central service that manages all Tag lifecycle (read, write, change detection, distribution) |
| Alarm Manager | Alarm Manager | Service that monitors Tag values against thresholds and manages alarm states |
| Recipe | Recipe | A preset set of Tag values that can be loaded/saved |
| HMI | Human-Machine Interface | Operator interface for industrial control systems |
| SCADA | Supervisory Control and Data Acquisition | Industrial control and monitoring system |
| WebSocket | WebSocket | Full-duplex communication protocol over a single TCP connection, used for real-time data push |
| Node.js | Node.js | JavaScript runtime built on Chrome's V8 engine, FUXA's backend platform |
| Angular | Angular | Frontend framework for building SPA applications, FUXA's frontend stack |
| socket.io | socket.io | WebSocket library for Node.js, used for real-time bidirectional communication |
| SQLite | SQLite | Lightweight embedded relational database, FUXA's default config storage |
| InfluxDB | InfluxDB | Time-series database for high-performance historical data storage |
| MQTT | Message Queuing Telemetry Transport | Lightweight publish/subscribe messaging protocol |
| OPC UA | Open Platform Communications Unified Architecture | Industrial communication standard for interoperability |
| Modbus | Modbus | Industrial serial communication protocol for PLC and sensor communication |
| BACnet | Building Automation and Control Networks | Protocol for building automation systems |
| EtherNet/IP | EtherNet/IP | Industrial Ethernet protocol by Allen-Bradley/Rockwell |
| JSON | JavaScript Object Notation | Text-based data interchange format used for FUXA project files |
| Docker | Docker | Containerization platform for packaging and deploying applications |
| Raspberry Pi | Raspberry Pi | Low-cost single-board computer suitable for edge SCADA deployment |
