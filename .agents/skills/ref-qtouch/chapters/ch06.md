# ch06 — 产品特色

## 6.1 高性能 HMI（High Performance HMI）

iFIX 最标志性的特色，基于 ISA 101 标准：

- **异常驱动显示**: 正常状态不显示（灰色/低饱和度），异常时突出
- **减少认知负荷**: 操作员可快速识别异常
- **500+ Dynamo 预置对象**: ISA 标准符号，拖拽即用
- **模型化导航**: Plant Model 层级导航（概览→区域→详细）

## 6.2 开放架构

- 基于标准技术：OPC, ActiveX, COM, VBA, ODBC
- 与 Office 集成，嵌入 Word/Excel 文档
- 第三方 ActiveX 控件嵌入
- OPC Toolkit 可开发私有 I/O 驱动
- REST API, SQL 数据库访问

## 6.3 分布式联网

- 按需数据访问（仅传输请求数据）
- 去中心化处理（SCADA 节点独立运行）
- 报警跨节点路由
- 网络隔离（实时数据网与同步网分离）

## 6.4 工业级可靠性

- 故障转移（主/备服务器自动切换）
- 数据库同步压缩（大标签库快速同步）
- 热备（备机实时同步，无缝切换）
- 安全模式（限制画面从可信位置加载）

## 6.5 厂商无关性

- 支持 200+ PLC 和驱动协议
- 不绑定任何硬件厂商
- 适合混合厂商环境

## 6.6 长期产品延续性

- 自 1983 年首发，超过 40 年
- 向后兼容性（新版本可打开旧项目）
- 庞大的安装基础

## 6.7 QTouch 差异化特色

- 跨平台技术栈（Qt），一次编程多平台适配
- 国产芯片支持（龙芯/飞腾/兆芯/鲲鹏/申威）
- IEC 61131-3 SoftPLC + C 语言混合编程
- 云网端边四维一体架构
- 嵌入式设备部署能力
