# ch03 — 功能概览

## 3.1 画面开发与可视化

- 基于向量的图形引擎
- 500+ Dynamo 预置对象（ISA 符号）
- 动态链接（主对象变更自动更新所有实例）
- 可视化类型：实时趋势、历史趋势、柱状图、面积图、数据表、3D
- QTouch: Qt Quick/QML 声明式 UI

## 3.2 SCADA 监控与控制

- 多协议并行数据采集
- 标签数据库管理
- 设定值写入、远程控制
- 批次控制、顺序控制、联锁逻辑
- QTouch: IEC 61131-3 SoftPLC + C 语言混合

## 3.3 报警管理

流程：检测 → 优先级分配 → 路由 → 操作员确认 → 记录 → 归档
类型：过程报警、系统报警、操作员消息、应用消息

## 3.4 历史数据管理

- Proficy Historian: 高性能时序采集、压缩存储、快速检索
- 免费归档：1,000 I/O 点的 Historian Essentials
- 云原生/本地部署选项

## 3.5 报表与分析

- 预配置报表模板、自定义报表设计
- 趋势分析、SPC 图表、OEE
- CSense 嵌入式分析引擎

## 3.6 用户管理

- 角色定义（Operator, Engineer, Administrator, Manager）
- 权限分级、LDAP/AD 集成
- 操作审计日志

## 3.7 Web 与移动端

- **Proficy Webspace**: HTML5 零部署客户端，全功能 SCADA 访问
- **Proficy Operations Hub**: 低代码/无代码 Web 应用开发
- **QTouch**: 云网端边一体化架构

## 3.8 关键工作流

### 工厂 SCADA 部署
1. 安装 KEPServerEX 配置 PLC 连接
2. 安装 iFIX WorkSpace
3. 创建标签数据库
4. 设计 HMI 画面
5. 配置报警和事件
6. 配置 Historian
7. 部署到 SCADA 服务器
8. 配置 Webspace 远程访问

### 多站点集中监控
1. 各站点部署 CIMPLICITY Server
2. 中央监控室部署 CIMPLICITY Client
3. 配置跨站点数据同步
4. 统一报警管理

## 3.9 扩展机制

- VBA 脚本（主要扩展方式）
- Configuration Hub（Web 集中配置）
- Integration Toolkit（ERP/MES 集成）
- OPC 工具包（自定义 I/O 驱动开发）
- QTouch: JS 脚本 + C 语言编译
