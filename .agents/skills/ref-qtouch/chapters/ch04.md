# ch04 — 现状与生态

## 4.1 版本与活跃度

- **iFIX 2023 / CIMPLICITY 2023**: 约每年一个主要版本
- **2023 版本更新**: 集成安装（15 分钟）、MQTT 客户端、Configuration Hub、安全模式、报警队列 10,000
- **2025**: iFIX 持续更新，Operations Hub 深度集成，嵌入式 CSense
- **QTouch**: 持续迭代，聚焦国产化生态建设

## 4.2 用户基数

- **iFIX/CIMPLICITY**: 20,000+ 组织全球（GE Vernova 官方数据）
- 行业覆盖：制药、水处理、电力、石油天然气、食品饮料、汽车、基础设施
- 地理分布：北美（核心）、欧洲、亚太、中东
- **QTouch**: 军工、国网、研究院等国产化替代市场

## 4.3 生态系统

- **OEM 合作伙伴**: KEPServerEX 200+ 驱动，Rockwell, Siemens, Schneider, Emerson
- **系统集成商**: 全球认证集成商网络
- **培训**: 官方培训课程、在线文档、专业服务团队

## 4.4 最新发展趋势

### 所有权变迁：GE → Emerson → GE Vernova → Velotic

| 时间 | 事件 |
|------|------|
| 1981 | Intellution 创立 |
| 1995 | Emerson 收购 ($100M) |
| 2001 | GE 从 Emerson 获得该业务 |
| 2024 | GE 分拆，成立 GE Vernova |
| 2026.03 | TPG 收购，成立 Velotic |

### 2026 年 3 月关键事件
- TPG 从 GE Vernova 收购 Proficy 软件业务
- 同时收购 PTC 的 Kepware 和 ThingWorx 业务
- 合并成立 **Velotic**（独立工业软件公司）
- KEPServerEX 与 Proficy 更紧密集成
